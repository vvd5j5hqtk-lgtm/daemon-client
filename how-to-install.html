INSTALLATION GUIDE

Follow these quick steps to set up Daemon Client correctly.

1

Install Tampermonkey Extension

First, you must install the official Tampermonkey extension for your browser. This extension allows the client script to run.

Get Tampermonkey
2

Enable Developer Mode

Go to your browser's extensions page (usually chrome://extensions/) and toggle "Developer mode" ON at the top right corner.

3

Set Config Mode to Advanced

Open the Tampermonkey dashboard, click on the "Settings" tab. Look for the "Config mode" option and change it from "Novice" to "Advanced".

4

Update Security Settings

Scroll down to the "Security" section. Find the "Content Script API" option and change it to UserScript to allow the mod to inject properly and save that.

You are all set!

Your browser is now ready. You can safely install the Daemon Client mod.
